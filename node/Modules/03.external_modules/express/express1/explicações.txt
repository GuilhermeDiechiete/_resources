CONSTRUÇAO DO PROJETO 

    1 - criar uma pasta  para o projeto 
    2 - npm init -y -> criar o package.json
    3 - npm install express 
    4 - express nomePastaProjeto --view=pug -> criar a arquitetura do projeto 
    5 - fazer os comandos que o express mandar

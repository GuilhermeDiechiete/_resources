## BASIC COMMAND LINUX 

### Ver o diretorio atual

```
pwd
```

### Listar arquivos do diretorio atual

```
ls
```

### Lista o conteudo de um arquivo 

```
cat
```

### Avançar ou voltar um diretorio

Avançar um diretorio
```
cd DiretoryName
```

Voltar um diretorio
```
cd ..
```

### Remover arquivos

```
rm fileName.txt
```

### Create File or Diretory

```
mkdir diretoryName
```

```
mkdir fileName.txt
```


TIPOS NUMERICOS 

TIPO            APLICAÇÂO

smallint            Inteiros de 2 bytes
int                 Inteiros de 4 bytes 
bigint              Inteiros de 8 bytes 

numeric             números de ponto flutuante, onde:precisão = numero de
                    digitos total e escala = n de cadas decimais

real                32 bits, até 6 digitos decimais de precisão
double precision    64 bits - variavel, até 15 digitos de precisão 
serial              32 bits de tamanho, com sinal, numeros seuenciais
big serial          64 bits de tamanho, com sinal, numeros seuenciais
money               64 bits, com sinal(2^63 valores).Para valores monetários


TIPOS DE STRING

text                    varchar de tamanho ilimitado. Tipo preferido para string.
char(n), character(n)   caracteres de tamanho fixo, com padding(preenchimento)
                        e n caracteres
varchar(n)              varchar de tamanho limitado até n caracteres


TIPOS DE DATA E HORA

date                    4 bytes, apenas datas, precisão de 1 dia 
time [without time zone]    8 bytes, hora sem fuso horario, com precisão de 1 microssegundo
tome with time zone     12 bytes, armazena data e hora com fuso horario, precisão de 1 microssegundo
timestamp with time zone 8 bytes, armazena data e hora com fuso horario, precisao de 1 microssegundo
timestamp [without time zone] 8 bytes, armazena data e hora sem fuso horario, precisao de 1 microssegundo
interval            1 bytes, armazena faixas de tempo, com precisao de 1 microssegundo


OUTROS TIPOS    

boolean         Tipo logico, 8 bits (1 byte) - Valores True(1,yes,on) ou False (0,no,off)

cidr            7 ou 19 bytes - endereços de rede IPv4 ou IPv6, como 192.168.14.0/14
inet            7 ou 19 bytes - endereços de hosts IPv4 ou IPv6, como 192.168.14.22/32

macaddr         6 bytes (48 buts), como 00:22:33:44:55:b2 ou 0022.3344.55b2

Geometric Types Armazenar informações relacionadas com figuras geometricas, como linhas, circulos, poligonos, pontos, caminhos, etc...

tipos de Enumeração     Criados pelo usuário, para conjuntos de valores estáticos.

tsvector / tsquery      Tipos para busca completa de texto em documentos.
